# C SDL Game

Just a bit of fun making a retro game using C with SDL

## Deps

You'll need to install the following:

- brew install SDL2
- brew install SDL2_image
- brew install SDL2_ttf

## Running the game

Compile with `make`
Then run the game with `./build/game`

## Project structure

```
game/
├── assets/
│ └── OpenSans_Condensed-Regular.ttf
├── src/
│ ├── main.c
│ └── screens/
│     └── start.c
├── include/
│   └── start.h
└── build/
    └── game
```
